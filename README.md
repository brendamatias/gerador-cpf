# [Gerador de CPF](https://github.com/brendamatias/desafios-frontend/blob/main/projects/beginner/Gerador-CPF.md) 🧾

**Nível:** 1-Iniciante

Neste desafio, você irá desenvolver uma aplicação simples de geração de CPF. O usuário poderá optar por gerar o CPF com ou sem pontuação e também terá a opção de copiar o número gerado.

## Design

- **Design do Projeto:** [Figma](https://www.figma.com/community/file/1205352885345206669)

## Histórias de Usuário

- [x] O usuário pode gerar um número de CPF ao clicar no botão "Gerar CPF".
- [x] O usuário pode escolher se deseja que o CPF seja gerado com ou sem pontuação.
- [x] O usuário pode copiar o CPF gerado para a área de transferência ao clicar no botão "Copiar".
- [x] O usuário pode ver o número de CPF atualizado no display conforme as suas opções selecionadas.

## Funcionalidades Adicionais

- [x] O usuário pode visualizar uma mensagem de sucesso ao copiar o CPF.
- [ ] O usuário pode selecionar quantos CPFs deseja gerar ao mesmo tempo.
- [ ] O usuário pode escolher entre temas claro e escuro para a interface.
